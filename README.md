# Weather App

TS + React.  
<br />
Weather app. You can find out the current weather in your city and other cities.  
<br />
Приложение погоды. Вы можете узнать текущую погоду в вашем городе и других городах.  
<br />
Link: https://currentweatherdata.netlify.app/

## How to start project

in the project directory enter:

```js
npm install
```

and then

```js
npm run dev
```
